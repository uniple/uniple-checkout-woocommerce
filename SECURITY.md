# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in uniple checkout for WooCommerce,
please email support@uniple.io with details. Do not file a public GitHub issue
for security reports.

We will acknowledge your report within 5 business days and provide a remediation
timeline based on the severity.

## Supported Versions

| Version | Supported |
|---|---|
| 0.1.7 | ✓ |
| < 0.1.7 | ✗ (please upgrade) |
